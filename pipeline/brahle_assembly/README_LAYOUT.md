INSTALACIJA
===========

Potrebni alati:
---------------

1. `g++ 4.8` ili noviji


Kompilacija:
------------

1. `make` 


Pokretanje
----------

1. `bin/main_layout <.afg file with reads> <.afg file with overlaps>`

Opcionalno:
-----------

1. `make test` - pokrece testove

2. `make valgrind-test` - provuce testove kroz valgrind, za detekciju memory
leakova

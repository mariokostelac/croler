Mile assembly
=============

Instalacija
-----------

    git submodule update --init --recursive
    ./make.sh

Rikvajrmenti
------------
(molim nekog da prevede ovo, ne mogu se sjetit hrvatske rijeci)
- gcc (4.7.2 na debianu fino kompajlira)
- make
- zlib (dev verzija - treba nam source)


Pokretanje
----------

    ./run.sh <reads.afg>
